## How to use
- Clone the repository
- run `./pymodoro.py`
- press 'q' to kill

```
optional arguments:
  -h, --help     show this help message and exit
  -w W           Focus/work minutes. (default: 25m)
  -b B           Break minutes. (default: 5m)
  -s S           Wav file to use for the notifications. (default: ./oi.wav)
  ```
