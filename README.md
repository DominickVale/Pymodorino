Very early development version. I'm just learning python, please be patient with my skills at this time :)

## How to use
- Clone the repository
- run `./pymodoro.py`

```
positional arguments:
  start        Start timer

optional arguments:
  -h, --help   show this help message and exit
  -w 25        Working minutes
  -b 5         Break minutes
  ```
